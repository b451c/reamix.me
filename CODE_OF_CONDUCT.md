# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

The full text is available at:
**https://www.contributor-covenant.org/version/2/1/code_of_conduct/**

## Scope

This Code of Conduct applies within all project spaces — the GitHub repository (issues, pull requests, discussions, code), the project's communication channels, and when an individual is officially representing the project in public spaces.

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by contacting the project maintainers via a private GitHub issue or email. All complaints will be reviewed and investigated promptly and fairly.

Project maintainers are obligated to maintain confidentiality with regard to the reporter of any incident.
